# indexMerge
This is a super small HTML index merger! Just put your indexes in the textarea and the result will pop.

The format it will merge your indexes:

Index1
--Surname1
--Surname2
--Surname3
--Surname4

Index2
--Yeah1
--Yeah2

Result:

Index1, Surname1
Index1, Surname2
Index1, Surname3
Index1, Surname4
Index2, Yeah1
Index2. Yeah2
